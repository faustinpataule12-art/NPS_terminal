# NPS_terminal — COFFRE

Application Android WebView sécurisée — Coffre-fort privé NPS.

## Description
**COFFRE** est une application Android qui charge une interface web sécurisée (VIE PRIVÉ — NPS) via WebView. Elle permet de stocker des fichiers de manière chiffrée avec un PIN local.

## Build
```bash
./gradlew assembleDebug
```
